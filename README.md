Personal Event Planner App

## About the Project
The Personal Event Planner is a native Android application designed to help users organize upcoming events, trips, or appointments. Built using modern Android architecture, it ensures all scheduling data is persistently stored on the device and provides a smooth, single-activity user experience.

## Key Features
**Create:** Add new events with a Title, Category (Work, Social, Travel), Location, and Date/Time.\
**Read:** View a dashboard of all upcoming events, automatically sorted chronologically.\
**Update:** Click on any existing event to modify its details.\
**Delete:** Remove events you no longer wish to track.\
**Input Validation:** Prevents saving events with empty fields or dates/times set in the past.\
**User Feedback:** Utilizes Toasts to confirm successful saves, updates, deletions, and validation errors.

## 🚀 How to Run the App
1. Clone this repository to your local machine.
2. Open the project in Android Studio.
3. Sync the project with Gradle files.
4. Build and run the application on an Android Emulator or physical device.
